### 命名变量使用帮助文档

#### 1. 命名变量概述

MediaMaster提供了丰富的命名变量，可用于自定义媒体文件和目录的命名规则。这些变量可以在系统设置中配置，支持电影、电视剧、动漫和综艺等多种媒体类型的命名。

#### 2. 可用命名变量

以下是可以用于目录命名的变量：

| 变量名 | 描述 |
|--------|------|
| `{title}` | 媒体标题 |
| `{title_en}` | 媒体英文标题 |
| `{year}` | 发布年份 |
| `{resolution}` | 分辨率 |
| `{tmdb_id}` | TMDB ID |


以下是可以用于文件命名的变量：

| 变量名 | 描述 |
|--------|------|
| `{title}` | 媒体标题 |
| `{title_en}` | 媒体英文标题 |
| `{year}` | 发布年份 |
| `{resolution}` | 分辨率 |
| `{season}` | 季数（电影不适用） |
| `{episode}` | 集数（电影不适用） |
| `{episode_title}` | 剧集标题（电影不适用） |
| `{video_codec}` | 视频编码 (如 H.264, H.265) |
| `{video_dynamic_range}` | 视频动态范围 (如 HDR, Dolby Vision) |
| `{source}` | 视频来源 (如 WEB-DL, BluRay) |
| `{audio_codec}` | 音频编码 (如 AAC, DTS) |
| `{audio_channels}` | 音频声道 (如 5.1, 7.1) |
| `{bit_depth}` | 位深度 (如 10bit) |
| `{extension}` | 文件扩展名 |
| `{tmdb_id}` | TMDB ID |
| `{original_filename}` | 原始文件名 |
| `{original_filename_no_chinese}` | 原始文件名（移除中文字符） |

#### 3. 条件格式支持

支持条件格式语法，可以根据变量是否存在来决定是否包含某些文本：

```
{variable?(format)}
```

例如：`{year?({year})}` 表示只有当年份存在时才会显示 `(年份)`。

#### 4. 配置项说明

在系统设置中可以单独设置以下命名规则：

##### 4.1 文件命名规则配置项

- 电影文件命名规则
- 电视剧文件命名规则
- 动漫文件命名规则
- 综艺节目文件命名规则

##### 4.2 目录命名规则配置项

- 电影目录命名规则
- 电视剧目录命名规则
- 动漫目录命名规则
- 综艺节目目录命名规则

#### 5. 使用示例

##### 5.1 电影命名规则示例

系统设置 **电影文件命名规则** 配置为：

```
{title} ({year}) [{resolution}][{video_codec}][{audio_channels}]
```

输出示例：

```
肖申克的救赎 (1994) [1080p][H.264][5.1]
```

##### 5.2 电视剧命名规则示例

系统设置 **电视剧文件命名规则** 配置为：

```
{title} - S{season}E{episode} - {episode_title} [{resolution}][{source}]
```

输出示例：

```
权力的游戏 - S01E01 - 凛冬将至 [1080p][WEB-DL]
```

##### 5.3 目录命名示例

系统设置 **电影目录命名规则** 配置为：

```
{title} ({year})
```

输出示例：

```
权力的游戏 (2011)
```

##### 5.4 条件格式示例

使用条件格式的命名：

```
{title}{year? ({year})} [{resolution}]
```

如果年份存在，输出：

```
肖申克的救赎 (1994) [1080p]
```

如果年份不存在，输出：

```
肖申克的救赎 [1080p]
```

#### 6. 注意事项

1. 变量使用花括号 `{}` 包围
2. 空值变量会被自动移除，其周围的空格也会被清理
3. 文件名会自动清理非法字符（如 `< > : " / \ | ? *`）
4. 文件名长度限制为200字符，目录名长度限制为100字符
5. 条件格式语法为 `{变量名?(格式)}`，只有变量存在且非空时才会显示指定格式
6. 支持嵌套的条件格式，但应避免过于复杂的嵌套结构
